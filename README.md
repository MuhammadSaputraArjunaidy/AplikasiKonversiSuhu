
# Aplikasi Konversi Suhu



# Deskripsi
aplikasi ini memungkinkan pengguna untuk mengkonversi suhu dari satu skala ke skala lainnya. Pengguna cukup memasukkan nilai suhu, memilih skala suhu awal dan tujuan, kemudian menekan tombol "Konversikan" untuk melihat hasil konversinya.
# Features

Berikut adalah deskripsi fitur-fitur utama aplikasi ini:

1. Kolom Input:

- Nilai Suhu: Kolom teks untuk memasukkan angka suhu yang ingin dikonversi.
- Pilih Skala Suhu: Dropdown menu yang memungkinkan pengguna memilih skala suhu awal, seperti Celsius, Fahrenheit, Kelvin, atau lainnya.

2. Pilihan Konversi:

- Ke Fahrenheit: Pilihan untuk mengonversi suhu ke skala Fahrenheit.
- Ke Celsius: Pilihan untuk mengonversi suhu ke skala Celsius.
- Ke Kelvin: Pilihan untuk mengonversi suhu ke skala Kelvin.
- Reamur: Pilihan untuk mengonversi suhu ke skala Reamur.

3. Tombol :

- Konversikan: Tombol yang digunakan untuk memulai proses konversi berdasarkan input suhu yang dimasukkan dan skala yang dipilih.

4. Hasil Konversi:

-  Konversi: Kolom teks untuk menampilkan hasil konversi suhu sesuai dengan pilihan yang telah ditentukan.

    private void konversiSuhu() {
    try {
        // Ambil nilai suhu dari input
        double suhu = Double.parseDouble(txtSuhu.getText());
        double hasil = 0;

        // Ambil skala suhu awal dari ComboBox
        String skalaAwal = (String) cmbSkala.getSelectedItem();

        // Cek arah konversi berdasarkan RadioButton yang dipilih
        if (rdbToFahrenheit.isSelected()) {
            if (skalaAwal.equals("Celcius")) {
                hasil = suhu * 9/5 + 32; // Celcius ke Fahrenheit
            } else if (skalaAwal.equals("Kelvin")) {
                hasil = (suhu - 273.15) * 9/5 + 32; // Kelvin ke Fahrenheit
            } else if (skalaAwal.equals("Reamur")) {
                hasil = suhu * 9/4 + 32; // Reamur ke Fahrenheit
            }
        } else if (rdbToCelcius.isSelected()) {
            if (skalaAwal.equals("Fahrenheit")) {
                hasil = (suhu - 32) * 5/9; // Fahrenheit ke Celcius
            } else if (skalaAwal.equals("Kelvin")) {
                hasil = suhu - 273.15; // Kelvin ke Celcius
            } else if (skalaAwal.equals("Reamur")) {
                hasil = suhu * 5/4; // Reamur ke Celcius
            }
        } else if (rdbToKelvin.isSelected()) {
            if (skalaAwal.equals("Celcius")) {
                hasil = suhu + 273.15; // Celcius ke Kelvin
            } else if (skalaAwal.equals("Fahrenheit")) {
                hasil = (suhu - 32) * 5/9 + 273.15; // Fahrenheit ke Kelvin
            } else if (skalaAwal.equals("Reamur")) {
                hasil = suhu * 5/4 + 273.15; // Reamur ke Kelvin
            }
        } else if (rdbToReamur.isSelected()) {
            if (skalaAwal.equals("Celcius")) {
                hasil = suhu * 4/5; // Celcius ke Reamur
            } else if (skalaAwal.equals("Fahrenheit")) {
                hasil = (suhu - 32) * 4/9; // Fahrenheit ke Reamur
            } else if (skalaAwal.equals("Kelvin")) {
                hasil = (suhu - 273.15) * 4/5; // Kelvin ke Reamur
            }
        }

        // Tampilkan hasil konversi
        lblHasil.setText("Hasil: " + hasil);
    } catch (NumberFormatException e) {
        JOptionPane.showMessageDialog(this, "Input harus berupa angka.", "Error", JOptionPane.ERROR_MESSAGE);
    }
}
